📊 Conversor Excel → JSON

Convierte tu archivo Excel de canciones a JSON para importar en la biblioteca

📋 PASO 1: Descargar plantilla Excel

La plantilla tiene las columnas necesarias para todas las canciones

📤 PASO 2: Subir tu archivo Excel (.csv)

📝 PASO 3: Copiar el JSON generado

📚 PASO 4: Importar en Biblioteca

1. Abre biblioteca.html

2. Haz clic en "📤 Importar Backup"

3. Selecciona el archivo JSON que descargaste

4. ¡Todas tus canciones aparecerán!
